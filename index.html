                Andy Tran

                , Frontend Developer with 4 years + experience with CSS

                Answered Feb 20, 2016

                If you've already learned HTML and CSS, what you can do is move on to learning the standards for each of them such as "BEMCSS", "SMACSS", "OOCCS" for CSS and taking a look into preprocessors like Sass, Less, Stylus, PostCSS for CSS. As for HTML, check out Jade, Handlebars, Haml.


                Anyways, what I would recommend is once a day, choose a website that you visit frequently and try to replicate it as close as possible. For example, if I were to visit Quora, I would try to create a website with as much similarity as possible.
